# Plugin Jeedom Estar Energy

Ce dépôt contient le plugin **Estar Energy** pour Jeedom. Il fournit la structure de base nécessaire pour débuter le développement de l'intégration Estar Energy :

- Configuration du plugin et des équipements personnalisés.
- Interface d'administration (bureau/mobile) prête à être adaptée.
- Squelettes PHP, JavaScript et modèles de widgets.
- Documentation utilisateur (dossier `docs/`).

Modifiez ces fichiers pour implémenter les interactions avec les services Estar Energy (API, synchronisation des données, etc.).
